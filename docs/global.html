IriString

In JSON-LD, an @id is an IRI (usually a persistent URL) used to uniquely identify a concept or thing. You can safely ignore this if you don't care about JSON-LD.

Example
http://identifiers.org/ncbigene/1234

JsonldContextObject

In JSON-LD, a @context adds semantic information to regular JSON. You can safely ignore this if you don't care about JSON-LD.

Stream

Node.js stream. This library additionally uses highland for easier stream handling, so you can use the highland methods on all streams returned.
